Rust Service Kit

2026-06-16 15:20

#Rust #Backend #Infra

项目简介

这个项目用来沉淀我在 Rust 后端服务开发里的通用能力，比如配置管理、错误处理、日志、数据库访问和部署规范。

目标

希望它最终能成为一套开箱即用的服务起步模板，而不是每次都从零搭工程骨架。

计划模块

  • 服务启动模板
  • 配置与环境变量管理
  • 统一日志与 tracing
  • 数据库连接与迁移
  • Docker 部署示例
